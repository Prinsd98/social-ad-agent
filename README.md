# Social Ad Campaign Agent  
A multi-agent system that generates, schedules, and evaluates social media ad campaigns.  
Built for the Google x Kaggle Agents Intensive Capstone (2025).

## Features
- Multi-agent architecture  
- Creative generation agent  
- Copywriting agent  
- Scheduler agent  
- Analytics agent  
- Memory system (learns from previous campaigns)  
- Tool abstraction  
- Simulated evaluation metrics
